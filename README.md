# Prediccion-de-Fuga-Churn-con-Machine-Learning
------------------------------------------------
## 📄 Descripción
Desarrollo de un modelo de clasificación para predecir qué usuarios tienen mayor probabilidad de cancelar su suscripción. Incluye preprocesamiento de datos, manejo de variables categóricas y evaluación de modelos.

## 🎯 Objetivos
- Realizar un Análisis Exploratorio de Datos (EDA) para entender la distribución de clientes y cargos mensuales.
- Implementar un modelo de clasificación (Random Forest) para predecir la probabilidad de fuga.
- Identificar las variables con mayor impacto en la retención (Feature Importance).
- Generar recomendaciones estratégicas basadas en la antigüedad de los clientes para reducir la tasa de abandono.

## 📊 Tecnologías y Librerías
- Python (3.x)
- Librerías principales: pandas, numpy, matplotlib, seaborn para manipulación y visualización.
- Machine Learning: scikit-learn para preprocesamiento (StandardScaler, LabelEncoder), entrenamiento del modelo y evaluación de métricas.

## ✅ Resultados Esperados
- Identificación de Riesgo: Capacidad para detectar clientes con alta probabilidad de fuga antes de que cancelen el servicio.
- Visualización del Desempeño: Matriz de confusión y reporte de clasificación que muestran la precisión del modelo en la Clase 1 (Churn).
- Insights de Negocio: Hallazgo clave sobre la importancia de la variable Tenure (antigüedad), indicando que los primeros 6 meses son críticos para la retención.
